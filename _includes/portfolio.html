Title Simon Game

Tools HTML, CSS, Javascript, jQuery

Description

A web app for the classic Simon Game:

  • The player is presented with a random series of button presses.
  • Player hears a sound that corresponds to each button both when the series of button presses plays, and when the player personally press a button.
  • Each time the player inputs the series of button presses correctly, it will replay the same series of button presses and an additional step.
  • The player will be notified if the buttons are wrong. If played in strict mode, if it is wrong button the game restarts at level 1, otherwise that series of button presses starts again to remind the player of the pattern so the player can try again.
